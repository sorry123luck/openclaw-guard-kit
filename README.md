# openclaw-guard-kit

Guard workflow kit for OpenClaw: guard executable, skill template, AGENTS rules, and one-command installers.

## Structure

- installer/ installers and update scripts
- skills/guard-flow/ OpenClaw skill
- 	emplates/ AGENTS append block and config examples
- src/guard/ guard source code
- eleases/ release notes and binary distribution notes

## Goal

Provide a lightweight guard workflow kit for OpenClaw:
- guard executable
- skill template
- AGENTS rule block
- one-command install/update/uninstall flow
